# NewsApp 📰

An Android application that fetches and displays the latest news headlines using the [NewsData.io](https://newsdata.io) API.

## Features

- Live news headlines from around the world
- News images loaded with Glide
- Tap any article to open it in the browser
- Clean RecyclerView-based UI

|------|---------|
| Android Studio | IDE |

| Gson | JSON parsing |
| Glide | Image loading |
| RecyclerView | News list UI |
| NewsData.io API | News data source |

## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/Tanvi2230/NewsApp.git
   ```
2. Open in Android Studio
3. Add your API key in `MainActivity.java`
   ```java
   String API_KEY = "your_api_key_here";
   ```
4. Run on emulator or device
